# Security Settings

## Two-Factor Authentication (2FA)

Protect your account by enabling 2FA. We support:
- **TOTP**: Use apps like Google Authenticator or Authy.
- **Email**: Receive a one-time code via email.

How to enable:
1. Go to **Settings**.
2. Select **Enable 2FA**.
3. Scan the QR code or verify your email.

## Passkeys

Experience passwordless login with Passkeys.

1. Go to **Settings** > **Passkeys**.
2. Click **Register New Passkey**.
3. Use your device's biometric authentication (fingerprint, face ID) or a security key.
